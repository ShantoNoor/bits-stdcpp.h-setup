# bits-stdc-.h-setup
## bits/stdc++.h

```c++
#include <bits/stdc++.h>
```

### bits folder locations: 
#### Android(termux):
```
usr/include/c++/v1/ 
```

#### Windows(Code Blocks):
```
MinGW\lib\gcc\x86_64-w64-mingw32\8.1.0\include\c++\
```

#### MacOS:
```
/Library/Developer/CommandLineTools/usr/include/c++/v1
```
```
/usr/local/include/bits
```

```shell
g++ -std=c++17 -DDEBUG -O3 stdc++.h
```
